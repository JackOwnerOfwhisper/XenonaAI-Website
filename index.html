Frequently Asked Questions

Have questions? We've got answers

Do I need a Groq API key?

Yes, XenonaAI uses the Groq API to access the Llama 3.3 70B model. You can get a free API key by signing up at console.groq.com.

Is Python required?

Yes, XenonaAI requires Python 3.8 or later. The installer will check if Python is installed and guide you to install it if needed.

Is my data private?

Your conversations are sent to the Groq API for processing, but they are not stored permanently. XenonaAI stores conversation history locally on your device for context, but this data never leaves your computer except when making API calls.

Can I customize the AI's behavior?

Yes, XenonaAI remembers your preferences about how it presents information. You can adjust settings through the interface and the AI will adapt to your preferences.

How much does it cost?

XenonaAI is free to download and use. You will need a Groq API key, which may have usage limits and costs depending on your usage volume.

Will there be updates?

Yes, we plan to release regular updates with new features and improvements. Check back on the website for the latest version.
